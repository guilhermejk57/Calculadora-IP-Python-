# Calculadora de IP em Python

Projeto de uma calculadora de redes feita em Python puro (sem bibliotecas externas).

## Funcionalidades
- Validação de IP
- Cálculo de rede e broadcast
- Suporte a CIDR
- Identificação de classe do IP
- IP público ou privado
- Quantidade de hosts e sub-redes

## Tecnologias
- Python
